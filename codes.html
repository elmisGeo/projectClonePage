Instructions

  1. After clicking the "Copy Code" button, open the file you want to edit immediately.
  2. In your file, click anywhere to select a text.
  3. Select all by pressing "Ctrl" + "A" to select all text in the code.
  4. Then paste. ("Ctrl" + "V")
  5. Lastly, save the file. ("Ctrl" + "S")
