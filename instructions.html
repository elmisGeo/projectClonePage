Downloading and Opening the Project

Step 1

After downloading the project file zip, locate the file in the downloads folder.

Step 2

After downloading the project file zip, extract the file in the downloads folder.

If a propmt appears, click "Extract".

Additional Information: You can move the extracted files to your desired location, which I highly reccommend. Just make sure you know where you placed it after. You can also leave it in the downloads folder.

Step 3

Open the extracted folder, and look for Project.sln file.

After that, right click the file and click "Open With" and select Microsoft Visual Studio.

Reminder: You may have a different version of Visual Studio, but as long as it is Visual Studio, it should work properly.

Reminder:If a "Security Warning" prompt appears, click Ok.

Open the extracted folder, and look for Project.sln file. After that, right click the file and click "Open With" and select Microsoft Visual Studio.

Changing Server Name

Step 1

After opening the project file, expand the project by clicking the arrow beside the vb icon of Project

Step 2

Double click the "databaseConnection.vb" to edit the file.

After opeing the file, look for the:
"Server=DESKTOP-0O55DTE\SQLEXPRESS;"

Since you will be raplacing the server name, clear the content inside the equal sign "=" and semi colon ";".

Step 3

If you dont know the name of your server, open Microsoft SQL Server Management Studio.

Copy the text inside the Server Name.

Step 4

After copying the server name, paste it in the databaseConnection.vb file.

Additional Stuff...

Database Name & Table

Make sure you also modile the "database" value in the databaseConnection.vb file.

Reminder: replace the database in the databaseConnection.vb file with the same name as the database in the Microsoft SQL Server Management Studio.

Also, make sure to have the same tables in the database:

  • LogIn_Table
  • Product_Table
  • ProductGroup_Table

Warning: The project won't work properly if any of these tables are not present or not similar to the examples on the right.

If you have done all the steps properly, you can now run the project.
